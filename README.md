# Psico & Calm - Sitio Profesional de Psicología

Sitio web adaptativo, optimizado y profesional diseñado para un consultorio terapéutico de psicología clínica y salud sexual, enfocado en transmitir calma, empatía y privacidad. Desarrollado como proyecto final para el curso de **Desarrollo Web** en Coderhouse.

## 🛠️ Tecnologías y Metodologías Utilizadas
* **HTML5**: Maquetación semántica y limpia estructurada en 5 secciones independientes.
* **SASS / SCSS**: Arquitectura modular de estilos mediante archivos parciales, variables globales para diseño escalable y anidamiento (*nesting*) avanzado.
* **Bootstrap 5**: Integración del sistema de grillas responsivas (`row`, `col`) y componentes dinámicos e interactivos (`Navbar` colapsable, `Accordion` para preguntas).
* **SEO & Accesibilidad**: Optimización On-Page con etiquetas meta description únicas por página, estructuración correcta de encabezados (`h1`-`h3`) y atributos de accesibilidad.
* **Control de Versiones**: Flujo de trabajo Git subido a repositorio remoto en GitHub y despliegue automatizado.

## 📂 Secciones del Sitio
1. **Inicio (`index.html`)**: Presentación del consultorio con propuesta de valor y llamada a la acción (CTA) principal.
2. **Servicios (`servicios.html`)**: Catálogo estructurado de las modalidades terapéuticas con tarjetas adaptativas.
3. **Nosotros (`nosotros.html`)**: Información institucional, perfil profesional y pilares de la metodología de trabajo.
4. **Preguntas Frecuentes (`faq.html`)**: Acordeón interactivo para resolver las dudas más comunes de los consultantes de forma dinámica.
5. **Contacto (`contacto.html`)**: Formulario maquetado y validado estructuralmente para la reserva de turnos.
